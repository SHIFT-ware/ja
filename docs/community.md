# **IaC活用研究会**
**IaC**[^1]を普及させることで、主にSIの労働生産性を向上するための活動を行います。
具体的にはSIにおいて以下のようなことを実現することを目指します。

{++

* **無駄な作業の排除**
* **長時間労働の削除**
* **構成情報の共有化**
* **ダブル・トリプルチェックの削減**
* **人的ミス（ヒューマンエラー）の削減**
* **その他、SIにおける生産性向上**

++}

このようなことを実現するため、**IaC活用研究所** を設立します。

Link : [コミュニティページへ](https://iacc.doorkeeper.jp/)

[^1]: Infrastructure as Code
